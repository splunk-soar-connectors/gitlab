GitLab SOAR App

This app connects a GitLab instance with Splunk SOAR instance. The GitLab App for SOAR allows playbooks to implement various actions from the GitLab API.
  • get users
  • get projects
  • get branches
  • create project trigger
  • list project triggers
  • trigger pipeline

Prerequisites

  • The version of the Splunk SOAR instance is running at least version 6.0.0
  • Access to a GitLab instance running at least version 11.11.3 and a user's access token
  • If using the trigger pipeline action, a GitLab runner has been deployed and registered to the relevant GitLab project
